# Solutions for Advent of Code 2022 puzzles

Puzzles available at https://adventofcode.com/2022/

## Run

Run all:
```
$ sbt run
```

Run selected day:
```
$ sbt run -d <day_number>
```

## Tests

Run unit tests:
```
$ sbt test
```

Run regression tests (check solution answers):
```
$ sbt Regression/test
```

## Used libraries
 * [zio](https://zio.dev/)
 * [scopt](https://github.com/scopt/scopt)
